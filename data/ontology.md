# Ontology

## Paper

The core type is defined in the Geo space.

Paper Type:
- Name (for all entities)
- Description (for all entities)
- Cover (for all entities)
- Abstract
- Authors (relation -> Person)
- Tags (relation -> Tag) 
- Web URL
- Related topics (relation -> Topic)
- Publish date 
- Published in (relation -> Project)
- Related spaces (relation -> Space)
